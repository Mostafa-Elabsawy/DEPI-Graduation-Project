## Literature Review  

### Feedback & Evaluation  
The proposed **Trendora e-commerce system** is evaluated based on its planned features and architecture. The system design aligns with modern e-commerce standards, including user authentication, product management, and secure checkout.
The use of Angular for frontend and ASP.NET Core for backend suggests a scalable and maintainable structure. The responsive design supports a good user experience across devices. However, since the system is not yet implemented, evaluation is limited to design quality rather than actual performance.

---

### Suggested Improvements(Optional features)

- Plan for performance optimization (efficient APIs and database queries)  
- Enhance UI/UX with clear navigation and user feedback  
- Consider adding features like reviews, wishlist, and order tracking
