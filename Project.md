# Trendora - E-Commerce Web Application

## Project Overview
**Trendora** is a modern, user-friendly e-commerce web application designed to provide a seamless online shopping experience. The platform allows users to browse a wide variety of products, add items to their cart, and complete secure transactions with ease. Trendora combines a clean, intuitive interface with powerful features to enhance customer engagement and satisfaction.

## Features
- **User Authentication**: Secure sign-up, login, and profile management.
- **Product Catalog**: Browse, search, and filter products by categories, brands, and price.
- **Shopping Cart**: Add, update, or remove products in the cart.
- **Checkout System**: Secure checkout with multiple payment options.
- **Order Tracking**: Track order status and history.
- **Admin Panel**: Manage products, categories, and user orders.
- **Responsive Design**: Fully responsive across desktop, tablet, and mobile devices.
- **Wishlist & Reviews**: Users can save favorites and write product reviews.
